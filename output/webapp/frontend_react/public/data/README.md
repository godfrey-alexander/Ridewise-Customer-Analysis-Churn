# Data files

Place the following CSV files here for the dashboard to load:

- **riders_trips.csv** — Trip data (from `../frontend/data/riders_trips.csv`)
- **rfm_data.csv** — RFM segment data (from `../frontend/data/rfm_data.csv`)

Copy from the Streamlit frontend:

```bash
cp ../../frontend/data/riders_trips.csv .
cp ../../frontend/data/rfm_data.csv .
```

Or copy from wherever your project stores these files.
